# Rian Negreiros

Como desenvolvedor .NET Core, sigo os princípios de SOLID, Clean Code e Clean Architecture em meus projetos, tenho experiência no desenvolvimento de aplicativos web e serviços usando a plataforma .NET Core, incluindo C#, ASP.NET Core e Entity Framework Core. Além disso, sou proficiente em desenvolvimento de APIs RESTful e familiarizado com padrões da indústria, como JSON e JWT. Possuo conhecimento em bancos de dados relacionais (PostgreSQL, SQL Server) e NoSQL (MongoDB, Redis).Além disso, tenho conhecimentos básicos em React e experiência em DevOps, uso de serviços de nuvem, como Azure e AWS e containerização com Docker.

Também sou tenho experiência em microsserviços com RabbitMQ para comunicação assíncrona e escalabilidade nesse contexto. Minha abordagem enfatiza a qualidade do código, eficiência e aderência a metodologias ágeis, como o Scrum. Estou sempre buscando aprender e me adaptar a novas tecnologias e ambientes de desenvolvimento para contribuir para o sucesso de projetos de software.
